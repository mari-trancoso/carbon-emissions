# README

Do you want to know how much carbon you emit?

Use this calculator to find out!

Simply provide:

* Emission source
* Quantity
* Dates (start and end)

## Pre-requisites

- Ruby 3.3.0
- Rails 7.1.4

## Setup the database

```bash
$ bin/rails db:prepare
```

## Running the application

```bash
$ bin/dev
```
